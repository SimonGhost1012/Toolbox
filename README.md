# ‚¨á Installation                                         
1. Download the Toolbox VX.X.zip file from the [Releases](https://github.com/SimonGhost1012/ToolBox/releases).

2. Extract the zip file wherever you want using [7-Zip](https://7-zip.com/download.html) or [WinRAR](https://www.win-rar.com/download.html).

3. Run Install-Update Requirements.bat so Toolbox runs properly.

4. Run the main file (it becomes an .exe file from version 5.0 onward).
